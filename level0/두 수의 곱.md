# 두 수의 곱
> 출처 : <https://school.programmers.co.kr/learn/courses/30/lessons/120804>

<br/>

#### 문제
정수 `num1`, `num2`가 매개변수 주어집니다. `num1`과 `num2`를 곱한 값을 return 하도록 solution 함수를 완성해주세요.

<br/>

#### 조건
- 0 ≤ `num1` ≤ 100
- 0 ≤ `num2` ≤ 100

<br/>

#### 입출력 예
| num1 | num2 | result |
| :--: | :--: | :----: |
|  3   |  4   |   12   |
| 27  |  19   |   513   |

<br/>

## 나의 풀이

```javascript
function solution(num1, num2) {
  return num1 * num2;
}
```
