# 양꼬치
> 출처 : <https://school.programmers.co.kr/learn/courses/30/lessons/120830>

<br/>

#### 문제
머쓱이네 양꼬치 가게는 10인분을 먹으면 음료수 하나를 서비스로 줍니다. 양꼬치는 1인분에 12,000원, 음료수는 2,000원입니다. 정수 `n`과 `k`가 매개변수로 주어졌을 때, 양꼬치 `n`인분과 음료수 `k`개를 먹었다면 총얼마를 지불해야 하는지 return 하도록 solution 함수를 완성해보세요.

<br/>

#### 조건
- 0 ≤ `n` ≤ 1,000
- n/10 ≤ `k` ≤ 1,000
- 서비스로 받은 음료수는 모두 마십니다.

<br/>

#### 입출력 예
| n | k | result |
| :--: | :--: | :----: |
| 10 | 3 | 124,000 |
| 64 | 6 | 768,000 |

<br/>

## 나의 풀이

```javascript
function solution(n, k) {
  return n * 12000 + (k - parseInt(n / 10)) * 2000;
}
```
